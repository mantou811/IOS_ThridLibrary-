常用第三方库GitHub地址

1. Alamofire: https://github.com/Alamofire/Alamofire
   Alamofire 是一个流行的 Swift 网络库，用于简化网络请求和数据处理。

2. Realm-Swift: https://github.com/realm/realm-cocoa
   Realm-Swift 是一个用于移动平台的数据库框架，提供了简单的 API 来处理本地数据存储。

3. SnapKit: https://github.com/SnapKit/SnapKit
   SnapKit 是一个用于 iOS 和 macOS 的自动布局库，它使用 Swift 的语法来简化界面布局的代码编写。

4. Kingfisher: https://github.com/onevcat/Kingfisher
   Kingfisher 是一个用于下载和缓存图片的 Swift 库，它提供了简单易用的 API 来加载和显示图片。
